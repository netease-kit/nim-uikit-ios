## 9.5.0(2023-04-20)
### New Features
* 系统通知合并显示，同类系统通知消息仅显示最新的一条
* 系统通知未读数聚合，通讯录页面显示所有未读数，验证页面显示每条消息的具体未读数（未读数为1时不显示数量），未读验证消息设置底色进行区分
* 同意好友申请后发送打招呼消息、添加好友页面实现自动刷新
* 验证消息新增过期时间：7d（10080s）
* contactRepo 新增 myFriends 接口，用于获取我的好友列表
* 顶部新增 topView，可添加自定义 view
* 支持自定义 cell 类型注册、解析

### Behavior changes
* 调整 "通讯录" 标题文案大小、左对齐
* 验证消息列表 UI 展示优化
* 验证消息页面不再显示群相关通知（群邀请不再需要同意）
* 多端登陆收到验证消息，一端已同意，另一端再拒绝，显示已处理文案并置消息状态为：已同意
* 无预设头像设置文字头像规则更改，统一取文字后两位：昵称 > uid（文字头像不取备注/群昵称）

### Bug Fixes
* 修复好友列表数据较多时不展示好友列表数据的问题
* 修复我的群聊列表没有即时刷新的问题
* 修复通讯录导航栏、好友选择列表导航栏跳转异常的问题

## 9.4.0(2023-03-08)
### New Features
* 添加好友页面输入框输入文字后新增一键清除按钮。
### Behavior changes
* 添加好友页面，好友同意后再次点击添加好友变成好友名片页，有发送消息和删除好友按钮
* 添加好友页面，输入框提示语优化: 输入查找用户id -> 请输入账号
* 通讯录右侧显示的首字母范围显示完整：["A"..."Z", "#"]
* 备注名长度限制为 15 字
### Bug Fixes
* 修复黑名单页好友昵称超长文案展示重叠的问题。
* 修复给好友添加备注输入空格也可以保存成功的问题。
* 优化通讯录好友列表排序规则。
* 修复加入黑名单 - 删除好友 - 添加好友 - 移除黑名单后对方不在通讯录显示的问题。
* 修复好友名片页面账号前无“账号：”，添加账号上方无颜色间隔的问题。
* 修复通讯录不同账号之间的头像相互复用的问题。
