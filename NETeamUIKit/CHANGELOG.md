## 10.8.2(2025-6-30)
### New Features
* 新增加入群组页面
* 新增群组详情页面
* 群管理新增 入群邀请需同意、入群审核 开关，默认关闭
* 新增群管理配置类 TeamKitConfigCenter，可配置创建高级群时的被邀请人同意入群模式和申请入群的模式
### Behavior changes
* 群设置页群名称下方展示群号，长按可复制
* 登录完成后群成员数据不再全量同步，进入群聊后针对单个群进行同步

## 9.5.0(2023-04-20)
### New Features
* 群聊设置页面新增 "标记" 列表，可查看所有标记的消息
### Behavior changes
* 创建群聊默认邀请不需要对方同意
* 创建群/邀请入群的人数限制设为公共常量
* 讨论组与高级群创建成功展示创建时间及创建成功通知（本地通知），其他通知消息均不展示发送时间

## 9.4.0(2023-03-08)
### Behavior changes
* 讨论组实现方案修改；由IMSDK的Normal实现，改为由高级群来实现。
* 讨论组和高级群的创建代码移到UI层中。
* 讨论组和高级群退出群聊时，清理群中消息提醒和通知的。
